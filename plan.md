Personal Portfolio Website for CSE Student (Class 2027)
Creating a modern, clean, and responsive personal portfolio website with a focus on SQL, Database Management, and Data Analytics.

User Review Required
IMPORTANT

Contact links (LinkedIn, GitHub) and Resume are placeholder. The user should replace them with their own.
Final year projects and academic projects are based on common CSE trends related to databases.
Proposed Changes
Tech Stack & Design
Core: HTML5, Vanilla CSS3, JavaScript.
Frameworks/Libraries: FontAwesome (icons), Google Fonts (Outfit/Inter).
Design Strategy: Soft slate-gray/indigo palette, responsive flexbox/grid layout, CSS animations (fade-in, hover).
File Structure
index.html: Main landing page structure.
styles.css: Full website styling, including responsive media queries.
script.js: Sticky navbar, scroll-to-section animations.
Components
Hero: Clean intro with name and dynamic tagline.
About Me: Brief intro and career aspirations.
Skills: Interactive skill cards with icons (SQL, Python, Excel, etc.).
Projects: Grid display for database-intensive projects.
Certifications: List or card display for achievement highlights.
Contact/Footer: Clean contact form and social links.
Verification Plan
Automated Tests
Run axe-core via browser testing to ensure accessibility.
Verify responsiveness using Chrome DevTools device simulation (Mobile, Tablet, Desktop).
Manual Verification
Check all links and internal anchors.
Verify smooth-scroll functionality.
Ensure visual consistency across major browsers (Chrome, Edge, Firefox).
